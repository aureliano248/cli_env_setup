# Shell Module

This module owns zsh, Oh My Zsh, and login shell integration.

- `oh_my_zsh.sh` installs Oh My Zsh and the configured zsh plugins.
- `profile.sh` chooses the login profile file and writes managed zsh/profile blocks.

Keep generated shell snippets here so profile behavior is easy to review. Profile snippets should point at the selected reusable or bootstrap-installed paths.
